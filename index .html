1สร้าง Google Spreadsheet
ไปที่ sheets.google.com → สร้างใหม่ → ตั้งชื่อ BorrowTracker
📋 Sheet ชื่อ Items คอลัมน์:
id · name · icon · borrower · borrowFrom · borrowTo · pendingReturn · pendingStaff
📋 Sheet ชื่อ Logs คอลัมน์:
timestamp · type · detail · staff
